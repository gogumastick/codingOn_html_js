overflow 속성- visible(default)

요소 박스에 내용이 긴 경우 어떻게 표현할지 선택하는 속성 - visible : 자식 요소가 부모요소의 범위를 넘어가도 그대로 보여줌 - scroll : 스크롤 바가 추가되어 스크롤 가능 - hidden : 부모 요소의 범위를 넘어가는 자식 요소의 부분 미표현 - auto : 내용의 넘어가지 않으면 그대로 표현 넘어가면 스크롤 구현
